# search__focus 

Foco no campo de Busca com desabilitação do restante da página com CSS Puro sem necessidade nenhuma de jQuery ou Vanilla

Solução para tornar os campos buscas com mais foco de atenção em websites, ao focar a busca é liberado uma mascara que desabilita o restante do conteúdo [da página] para [que] a atenção do usuário [possa] ficar restrita somente ao campo de busca.
